# VaultMind 学术标签受控词表 (Taxonomy)

## 1. 学科与技术主题 (Subject Tags)
*这些标签描述“研究了什么”。*

- **人工智能 (AI)**: 机器学习, 深度学习, 强化学习, 神经网络, 计算机视觉, 自然语言处理 (NLP), 语音识别, 知识表示, 专家系统。
- **架构与模型**: Transformer, CNN, RNN, GAN, 大语言模型 (LLM), 扩散模型, 注意力机制, 编码器-解码器。
- **数据与特征**: 数据挖掘, 特征工程, 数据增强, 标注数据, 向量空间, 知识图谱。
- **优化与训练**: 损失函数, 梯度下降, 过拟合, 迁移学习, 微调 (Fine-tuning), 剪枝, 量化。
- **应用领域**: 医疗 AI, 自动驾驶, 机器人, 生物信息学, 金融科技, 智能制造。

## 2. 文献属性与方法论 (File/Method Tags)
*这些标签描述“是什么类型的研究”。*

- **研究类型**: 理论研究, 实验研究, 综述论文, 方法论, 案例研究, 比较研究, 复制研究。
- **数据属性**: 定量研究, 定性研究, 混合方法, 模拟数据, 真实世界数据。
- **学术环节**: 问题定义, 文献综述, 模型设计, 实验验证, 结果分析, 局限性讨论。

## 3. 同义词归一规则 (Normalization Rules)
*模型在输出前应检查并映射到标准标签。*

- "Attention", "Attention Mechanism", "Self-Attention" -> **注意力机制**
- "LLM", "Large Language Model", "GPT" -> **大语言模型 (LLM)**
- "Deep Learning", "DL" -> **深度学习**
- "Computer Vision", "CV" -> **计算机视觉**
- "Natural Language Processing", "NLP" -> **自然语言处理 (NLP)**
- "Transfer Learning" -> **迁移学习**
- "Fine-tuning" -> **微调 (Fine-tuning)**
